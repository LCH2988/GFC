🔵 基本運行模式

正常運轉模式 Normal

電梯的標準自動運行模式，響應內外召呼車請求[[235]](#__235)

📋 模式特徵：

  • 參數 00-09 bit 3 = 1 表示正常模式
  • 自動響應內召和外召呼車
  • 執行集選運行邏輯
  • 門控自動開關

🔧 相關參數：

00-09 bit 3 03-27 集選方式

多段速運行模式 Multi-Speed

根據不同運行階段使用不同速度[[234]](#__234)

📋 速度階段：

  • 檢修速度：參數 04-01 (0.25 m/s)
  • 平層內速度：參數 04-05 (0.04 m/s)
  • 平層外速度：參數 04-04 (0.12 m/s)
  • 返端站速度：參數 04-02 (0.50 m/s)
  • 井道學習速度：參數 04-03 (0.20 m/s)

🔧 相關參數：

04-01~04-05 00-01 速度指令

直接停靠模式 Direct Stop

使用 S 曲線加減速直接停靠目標樓層[[230]](#__230)[[231]](#__231)

📋 速度曲線參數：

  • 加速度：參數 04-07 (0.40 m/s²)
  • 減速度：參數 04-08 (1.50 m/s²)
  • S1 加速起始：參數 04-14 (3.00 秒)
  • S2 加速到達：參數 04-15 (1.00 秒)
  • S3 減速起始：參數 04-16 (1.00 秒)
  • S4 減速到達：參數 04-17 (3.00 秒)

🔧 相關參數：

04-07~04-17 13-20 最大梯速%

再平層模式 Re-Leveling

開門狀態下修正平層位置[[234]](#__234)

📋 觸發條件：

  • 參數 03-21 = 1 (可開門再平層)
  • 門已開啟且位置偏離平層
  • 使用參數 04-11 減速度

🔧 相關參數：

03-21 開門再平層 04-11 再平層減速度

🔴 安全運行模式

檢修運轉模式 Inspection

維護人員手動控制電梯運行[[222]](#__222)[[223]](#__223)

優先權等級：高

📋 檢修類型：

  • 控盤檢修：MI = 7 INSCP
  • 轎頂檢修：MI = 112 TCI
  • 轎廂內檢修：MI = 153 INSIC
  • 底坑檢修：MI = 10 MATPT

🎯 優先權順序：

  • 底坑檢修 = 轎頂檢修 > 轎廂內檢修 > 控盤檢修 > 緊急電動

⚙️ 速度控制：

  • 前 3 秒：參數 04-05 平層內速度
  • 3 秒後：參數 04-01 檢修速度
  • 強迫減速時：參數 04-04 平層外速度

🔧 相關參數：

06-07~06-10 04-01 檢修速度 04-18 慢速時間 02-44 上行極限 02-45 下行極限

緊急電動運行 Emergency

停電或故障時的應急運行模式[[222]](#__222)[[223]](#__223)

優先權等級：中高

📋 啟動條件：

  • MI = 11 EIS (緊急電動運行) ON
  • MI = 12 UDB (上行按鈕) 或 MI = 13 DDB (下行按鈕)

⚙️ 速度控制：

  • 前 3 秒：參數 04-05 平層內速度
  • 3 秒後：參數 04-04 平層外速度

🔧 相關參數：

06-11~06-13 04-04 平層外速度 04-18 慢速時間

強制停止減速 (FSD) FSD

安全迴路異常時的強制減速停止[[235]](#__235)

優先權等級：最高

📋 觸發條件：

  • 安全迴路斷開 (SFH/SFL/SFNC)
  • 門鎖迴路異常 (GSH/DSH)
  • 限速器動作 (GOV)
  • 上/下限位動作 (LSU/LSD)

⚙️ 減速方式：

  • 使用參數 04-29 檢修減速度 (2.0 m/s²)
  • 強制煞車閉合
  • 速度指令 = 5 或 10

🔧 相關參數：

00-09 bit 2 04-29 檢修減速度

救援運轉模式 Rescue

困人時的救援運行[[235]](#__235)

優先權等級：高

📋 救援類型：

  • UPS 救出：參數 00-08 bit 9
  • EPS 救出：參數 00-08 bit 10
  • 地震高救出：MI = 64 ERGO

🎯 運行目標：

  • 運行至最近平層開門
  • 或運行至指定樓層 (參數 03-09)

🔧 相關參數：

00-08 bit 9~10 03-09 UPS 指定樓層 03-15 地震高救出

制動力檢測模式 Brake Test

自動檢測煞車制動力是否合格[[226]](#__226)

優先權等級：中

📋 測試類型：

  • 手動測試：參數 03-67 = 1
  • 自動測試：每天指定時間 (參數 03-64~03-66)

⚙️ 測試參數：

  • 持續時間：參數 03-60 (5.0 秒)
  • 輸出力矩：參數 03-61 (100%)
  • 移動距離：參數 03-62 (5.0 mm)
  • 測試方向：參數 03-63 (0=下 / 1=上)

🔧 相關參數：

03-60~03-67 00-01 = 6 或 8

過速度保護模式 Overspeed

強迫減速開關處的速度監控[[258]](#__258)[[259]](#__259)

優先權等級：最高

📋 監控點：

  • ULS1~ULS4：上強減 1~4 級
  • DLS1~DLS4：下強減 1~4 級
  • 預設保護值：額定速度的 50%

⚠️ 故障碼：

  • F194~F197：ULS1~ULS4 過速度
  • F198~F201：DLS4~DLS1 過速度

🔧 相關參數：

03-37~03-44

🟡 特殊運行模式

消防運轉模式 Fire Service

火災時的專用運行模式[[235]](#__235)[[236]](#__236)

優先權等級：最高

📋 消防階段：

  • 一階消防：參數 00-08 bit 11~12 (返基站)
  • 二階消防：參數 00-09 bit 8~9 (消防員操作)

🎯 基站設定：

  • 主消防基站：參數 03-49 (MI = 44 FIRM)
  • 次消防基站：參數 03-50 (MI = 45 FIRS)

⚙️ 運行特性：

  • 禁止外召響應
  • 返基站後開門 (參數 05-20 控制)
  • 二階時需持續按開門鈕 (參數 05-23 = 0)

🔧 相關參數：

03-49~03-50 05-20 關門控制 05-23 自動開門

地震運轉模式 Earthquake

地震感應時的安全運行[[235]](#__235)

優先權等級：高

📋 地震等級：

  • 地震特低 (P 波)：MI = 51 ERP，參數 00-08 bit 13
  • 地震低 (S 波)：MI = 52 ERS，參數 00-08 bit 14
  • 地震高：MI = 53 ERH，參數 00-08 bit 15

⚙️ 運行策略：

  • P 波/S 波：減速至最近平層停止
  • 地震高：立即停止，可選擇自動救出 (參數 03-15)
  • 減速時間：參數 03-19 (7.0 秒)

🔧 相關參數：

03-15 地震高救出 03-19 減速時間

水災運轉模式 Flood

水災時避開淹水樓層[[235]](#__235)

優先權等級：中

📋 水災類型：

  • 不自動復歸：MI = 48 FLOOD，參數 00-08 bit 7
  • 會自動復歸：MI = 49 FLOODA，參數 00-08 bit 6

⚙️ 運行限制：

  • 電梯運行至水災基站 (參數 03-52)
  • 不停靠水災基站以下樓層
  • 水災基站預設為 2F

🔧 相關參數：

03-52 水災基站

鎖梯模式 Park

暫停服務並返回指定樓層[[235]](#__235)[[236]](#__236)

優先權等級：中

📋 啟動方式：

  • MI = 43 PARK (鎖梯開關)
  • 參數 00-09 bit 10 = 1

🎯 鎖梯基站：

  • 參數 03-48 設定 (預設 1F)
  • 0 = 取消鎖梯功能
  • 1~64 = 前門
  • 101~164 = 後門

⚙️ 門控制：

  • 參數 05-19 = 0：保持開門
  • 參數 05-19 = 1：開門 20 秒後關門

🔧 相關參數：

03-48 鎖梯基站 05-19 關門控制

VIP 專用模式 VIP

專人專用電梯服務[[236]](#__236)

優先權等級：中

📋 啟動方式：

  • MI = 164 VIP (VIP 開關)
  • 參數 00-10 bit 0 = 1

⚙️ 運行特性：

  • 電梯返回 VIP 層待命 (參數 03-33)
  • 待命時間：參數 03-32 (4 秒)
  • 禁止外召響應
  • 內召限制：參數 03-34 控制

🔧 相關參數：

03-32~03-34

司機操作模式 Attendant

司機手動控制電梯運行[[236]](#__236)

優先權等級：中

📋 啟動方式：

  • MI = 165 ATS (司機模式開關)
  • 參數 00-10 bit 1 = 1

⚙️ 運行特性：

  • 禁止外召響應
  • 司機控制開關門
  • 司機選擇目標樓層

醫療召回模式 Medical

醫療緊急用途專用電梯[[236]](#__236)

優先權等級：高

📋 啟動方式：

  • MI = 179 SEV (醫療召回開關)
  • 參數 00-10 bit 13 = 1

⚙️ 運行特性：

  • 優先響應醫療召回
  • 禁止其他外召
  • 快速響應

保安層模式 Security

特定時段限制樓層訪問[[236]](#__236)[[258]](#__258)

優先權等級：中

📋 設定參數：

  • 保安層基站：參數 03-29 (0 = 關閉)
  • 開始時間：參數 03-30 (0~2359)
  • 結束時間：參數 03-31 (0~2359)

⚙️ 運行限制：

  • 時段內只能到達保安層
  • 需刷卡才能到其他樓層
  • 參數 00-10 bit 10 = 1

🔧 相關參數：

03-29~03-31

🟢 其他運行模式

井道自學習模式 Shaft Learning

自動學習井道參數和樓層位置[[224]](#__224)[[225]](#__225)[[241]](#__241)[[242]](#__242)

優先權等級：中

📋 執行步驟：

  • 1. 設定參數 03-01 物理最高層、03-02 物理最低層
  • 2. 設定參數 04-03 井道學習速度 (建議 0.25 m/s)
  • 3. 將檢修開關切到檢修位置
  • 4. 設定參數 03-00 = 1
  • 5. 將檢修開關切到自動位置
  • 6. 電梯自動下行至 DLS1 再上行至 ULS1
  • 7. 完成後自動停在頂樓

⚙️ 學習內容：

  • 各樓層平層位置 (參數 09、10 群)
  • 強迫減速開關位置
  • 限位開關位置

⚠️ 注意事項：

  • 學習過程中參數 00-63 顯示狀態碼
  • 學習中途不可切換檢修開關
  • 完成後需執行制動力測試 (參數 03-67)

🔧 相關參數：

03-00 井道自學習 03-01~03-02 04-03 學習速度 00-63 狀態碼

重上電定位模式 Re-Position

斷電後重新上電的位置確認[[236]](#__236)[[257]](#__257)

優先權等級：中

📋 定位方式：

  • 0：關閉功能
  • 1：重上電後行駛至底樓定位
  • 2：重上電後行駛至頂樓定位
  • 3：依目前位置自動決定向上或向下定位

⚙️ 執行條件：

  • 參數 00-10 bit 2 = 1
  • 編碼器位置資訊遺失
  • 需重新確認樓層位置

🔧 相關參數：

03-17 重上電定位

空閒返基站模式 Idle Return

無呼車時自動返回基站節能[[236]](#__236)[[257]](#__257)

優先權等級：低

📋 功能設定：

  • 空閒基站：參數 03-05 (預設 1F)
  • 空閒時間：參數 03-06 (0 = 關閉功能)
  • 照明關閉：參數 03-07 (600 秒)
  • 風扇關閉：參數 03-08 (30 秒)

⚙️ 運行邏輯：

  • 無呼車且超過設定時間
  • 自動返回基站
  • 關閉照明和風扇節能

🔧 相關參數：

03-05~03-08 00-10 bit 3

群控運轉模式 Group Control

多台電梯協同調度運行[[229]](#__229)[[236]](#__236)[[257]](#__257)[[258]](#__258)

優先權等級：中

📋 基本設定：

  • 群控站號：參數 03-03 (0~7)
  • 群控致能：參數 03-04 (0=單台 / 1=群控)
  • 主站設定：參數 03-68~03-74

⚙️ 調度策略：

  • 反向加樓層：參數 03-91 (4 層)
  • 停一次時間：參數 03-92 (14.0 秒)
  • 分散等待時間：參數 03-75

🎯 高峰運轉：

  • 上行高峰：參數 03-76~03-77，MI = 59 UPK
  • 下行高峰：參數 03-78~03-79，MI = 60 DPK
  • 任意樓高峰：MI = 181 PK

🔧 相關參數：

03-03~03-04 03-68~03-90 00-61 連線狀態

每樓停模式 Every Floor

自動停靠每個樓層[[236]](#__236)

優先權等級：低

📋 啟動方式：

  • MI = 158 ESS (每樓停開關)

⚙️ 運行特性：

  • 自動登錄所有樓層內召

獨立運行模式 Independent

脫離群控單獨運行[[229]](#__229)[[236]](#__236)

優先權等級：中

📋 脫離方式：

  • 1. 設定參數 03-04 = 0
  • 2. 拔除 CAR 通訊線
  • 3. MI = 161 ISS (獨立運行開關) ON

⚙️ 運行特性：

  • 參數 00-10 bit 9 = 1
  • 不受群控調度
  • 獨立響應呼車

搬貨專用模式 Cargo

貨物運輸專用設定[[236]](#__236)

優先權等級：低

📋 啟動方式：

  • MI = 162 IND (搬貨專用開關)
  • 參數 00-10 bit 14 = 1

⚙️ 運行特性：

  • 延長開門時間
  • 禁止外召響應
  • 適用於貨物運輸

密碼呼車模式 Password

特定樓層需密碼才能呼車[[236]](#__236)[[258]](#__258)

優先權等級：低

📋 設定方式：

  • 密碼設定：參數 03-45 (0~65535)
  • 參數 00-10 bit 12 = 1

⚙️ 使用方法：

  • MI = 183 PASSWORD (密碼呼車開關)
  • MI = 184 PASSSET (密碼設定開關)
  • 需輸入正確密碼才能登錄內召

🔧 相關參數：

03-45 密碼

自動測試模式 Auto Test

自動運行測試電梯功能[[255]](#__255)

優先權等級：低

📋 測試方式：

  • 0：不測試
  • 1：頂底樓測試

⚙️ 測試參數：

  • 測試方法：參數 02-03
  • 測試次數：參數 02-04 (65535 = 持續測試)
  • 間隔時間：參數 02-05 (300 秒)

🔧 相關參數：

02-03~02-05

門鎖旁路模式 Door Bypass

緊急情況下繞過門鎖檢測[[235]](#__235)

優先權等級：高

📋 啟動方式：

  • MI = 56 DBP (門鎖旁路開關)
  • 參數 00-08 bit 5 = 1

⚠️ 安全警示：

  • MO = 105 BPSL (聲光報警) 啟動
  • MO = 106 UDCL (轎底燈) 閃爍
  • MO = 104 BZ (蜂鳴器) 鳴叫
  • 參數 02-10 = 1 可禁止鳴叫

02-10 不鳴叫

UPS 供電模式 UPS

不斷電系統供電時的運行[[235]](#__235)[[236]](#__236)

優先權等級：高

📋 啟動條件：

  • MI = 67 UPS (UPS 供電信號)
  • 參數 00-08 bit 8 = 1 (停機)
  • 參數 00-08 bit 9 = 1 (救出)
  • 參數 00-09 bit 11 = 1 (UPS 模式)

⚙️ 運行策略：

  • 運行至指定樓層 (參數 03-09)
  • 或運行至最近平層

EPS 緊急電源模式 EPS

緊急電源供電時的運行[[35]](#__35)[[36]](#__36)[[235]](#__235)[[236]](#__236)

優先權等級：高

📋 啟動條件：

  • MI = 54 EPS (緊急電源信號)
  • 參數 00-08 bit 10 = 1 (救出)

  • DC bus 電壓依參數 13-48 顯示

🔧 相關參數：

13-48~13-49

一、基本系統設定

1.1 物理樓層設定[[220]](#__220)[[225]](#__225)

參數 03-01 物理最高層
• 設定範圍：1~64F
• 出廠值：1
• 說明：設定電梯服務的最高物理樓層
參數 03-02 物理最低層
• 設定範圍：1~64F
• 出廠值：1
• 說明：設定電梯服務的最低物理樓層
📌 設定範例：
若實際樓層為 B1F~10F，則：
  • 物理最低層 = 1（代表 B1F）
  • 物理最高層 = 11（代表 10F）
⚠️ 群控注意事項[[225]](#__225)[[226]](#__226)：
群控時需以所有電梯的最高最低物理樓層設定
範例 電梯 1 樓層 電梯 2 樓層 物理最低層 物理最高層
單控 B1F~9F 1F~10F 1 / 1 10 / 10
群控 B1F~9F 1F~10F 1 / 1 11 / 11

1.2 服務樓層設定[[247]](#__247)

參數 03-11 服務樓層 16~1
• 設定範圍：0~65535
• 出廠值：65535（全部服務）
參數 03-12 服務樓層 32~17
• 設定範圍：0~65535
• 出廠值：65535
參數 03-13 服務樓層 48~33
• 設定範圍：0~65535
• 出廠值：65535
參數 03-14 服務樓層 64~49
• 設定範圍：0~65535
• 出廠值：65535
📌 設定方式：使用 Bit 位元控制各樓層是否服務
  • Bit 0 = 1：服務該樓層
  • Bit 0 = 0：不服務該樓層

二、運行模式設定

2.1 集選方式[[247]](#__247)

參數 03-27 集選方式
• 設定範圍：0~3
• 出廠值：0
• 選項說明：
  • 0：無功能
  • 1：全集選（所有方向呼車都響應）
  • 2：上集選（只響應上行呼車）
  • 3：下集選（只響應下行呼車）

2.2 內外召控制[[245]](#__245)

參數 02-07 內外召禁能
• 設定範圍：0~5
• 出廠值：0
• 選項說明：
  • 0：正常運行
  • 1：外召禁能（只響應內召）
  • 2：外召禁能，內召無法再有新的叫車登錄，參數 02-00 內召測試有效
  • 3：內外召皆禁能
  • 4：內、外召全部禁能，且開門禁能
  • 5：外召禁能，內召無法再有新的叫車登錄，且開門禁能

2.3 誤按消除功能[[248]](#__248)

參數 03-28 誤按消除功能
• 設定範圍：0~1
• 出廠值：0
• 選項說明：
  • 0：無誤按取消功能
  • 1：有內召誤按消除（再按一次取消）

三、特殊運行模式

3.1 空閒返基站[[247]](#__247)

參數 03-05 空閒基站
• 設定範圍：1~64S
• 出廠值：1
• 說明：設定電梯空閒時自動返回的樓層
參數 03-06 空閒返基站時間
• 設定範圍：0~65535 秒
• 出廠值：0（關閉功能）
• 說明：電梯空閒超過此時間後自動返回基站
參數 03-07 空閒轎廂照明關閉
• 設定範圍：0~65535 秒
• 出廠值：600 秒
• 說明：空閒時自動關閉轎廂照明的延遲時間
參數 03-08 空閒轎廂風扇關閉
• 設定範圍：0~65535 秒
• 出廠值：30 秒
• 說明：空閒時自動關閉轎廂風扇的延遲時間

3.2 重上電定位[[247]](#__247)

參數 03-17 重上電定位
• 設定範圍：0~3
• 出廠值：0
• 選項說明：
  • 0：關閉功能
  • 1：重上電後行駛至底樓定位
  • 2：重上電後行駛至頂樓定位
  • 3：依電梯目前位置自動決定向下或向上定位

3.3 VIP 模式[[248]](#__248)

參數 03-32 VIP 待命時間
• 設定範圍：0~65535 秒
• 出廠值：4 秒
• 說明：VIP 模式啟動後的待命時間
參數 03-33 VIP 層
• 設定範圍：0~164
• 出廠值：0
• 選項說明：
  • 0：無功能
  • 1~64：指定該設定值為前門的 VIP 層
  • 101~164：指定該設定值減 100 為後門的 VIP 層
參數 03-34 VIP 內召限制
• 設定範圍：0~1
• 出廠值：0
• 說明：VIP 模式時是否限制內召叫車

五、群控功能設定

5.1 基本群控設定[[224]](#__224)[[247]](#__247)

• 設定範圍：0~7
• 出廠值：0
• 說明：設定該電梯在群控系統中的站號（不可重複）
參數 03-04 群控致能
• 設定範圍：0~1
• 出廠值：0
• 選項說明：
  • 0：單台運轉
  • 1：群控運轉
📌 脫離群控方法[[224]](#__224)：
  1. 將參數 03-04 設定為 0
  2. 拔除該台電梯的 CAR 通訊線
  3. 令輸入接點功能碼 161 ISS（獨立運行開關）端子為 ON

• 設定範圍：0~59

• 說明：每天自動執行煞車測試的分鐘
參數 03-66 幾層-煞車測試
• 設定範圍：0~64
• 出廠值：0
• 說明：執行煞車測試的指定樓層，0 表示在任何樓層

9.3 手動啟動測試[[226]](#__226)[[250]](#__250)

參數 03-67 開始-煞車測試
• 設定範圍：0~1
• 出廠值：0
• 操作說明：
  1. 先使電梯停在平層位置，並且電梯必須處於手動模式
  2. 確保沒有任何故障發生
  3. 設定參數 03-67 = 1，會開始制動力測試
  4. 制動力測試完成後，參數 03-67 將自動返回到 0
⚠️ 重要提醒：電梯第一次自動運行前一定要完成制動力測試，否則將出現故障代碼 F150（制動力檢測不合格）[[226]](#__226)

十四、快速設定指南

14.1 新電梯初次設定流程

🚀 步驟 1：基本樓層設定
  • 參數 03-01：物理最高層
  • 參數 03-02：物理最低層
🚀 步驟 2：門系統設定
  • 參數 05-00：門機數量（1=單門，2=雙門）
  • 參數 05-01：開門控制（1=先開前門，2=同時開）
  • 參數 03-51：ICB 功能設定
🚀 步驟 3：速度設定
  • 參數 04-01：檢修速度（建議 0.25 m/s）
  • 參數 04-03：井道學習速度（建議 0.25 m/s）
  • 參數 04-07：加速度
  • 參數 04-08：減速度
🚀 步驟 4：安全基站設定
  • 參數 03-05：空閒基站
  • 參數 03-48：鎖梯基站
  • 參數 03-49：主消防基站
  • 參數 03-52：水災基站
⚠️ 步驟 5：執行井道自學習
  • 參數 03-00 = 1
⚠️ 步驟 6：執行制動力測試
  • 參數 03-67 = 1

14.2 群控系統設定流程

🔗 步驟 1：各台電梯設定站號
  • 參數 03-03：群控站號（0~7，不可重複）
🔗 步驟 2：統一物理樓層
  • 所有電梯的參數 03-01、03-02 設定相同
🔗 步驟 3：啟用群控
  • 參數 03-04 = 1
🔗 步驟 4：設定群控主站（選配）
  • 參數 03-68~03-74：各樓層的群控主站
🔗 步驟 5：設定高峰運轉（選配）
  • 參數 03-76~03-83：高峰時段
  • 參數 03-84~03-89：高峰條件

14.3 常用功能快速設定

📌 延長開門時間

  • 參數 05-10：外召開門時間（預設 5.0 秒）
  • 參數 05-11：內召開門時間（預設 5.0 秒）
  • 參數 05-12：開門延時時間（預設 30.0 秒）

📌 調整電梯速度

  • 如果速度不準確，調整參數 13-03（電梯額定速度頻率）[[232]](#__232)
  • 直接停靠模式：調整參數 13-20（最大梯速百分比）[[232]](#__232)

📌 停車位置調整

  • 停車過頭：調小參數 04-08（減速度）或調大參數 04-17（S4 減速到達時間）[[232]](#__232)

📌 開啟誤按消除

  • 參數 03-28 = 1

📌 設定空閒返基站

  • 參數 03-05：空閒基站樓層
  • 參數 03-06：空閒返基站時間（秒）

  1. 所有參數設定完成後，建議使用參數複製功能（參數 00 群）備份設定[[176]](#__176)[[177]](#__177)
  2. 修改 MI/MO 接點設定後，需重新送電才會正常動作[[206]](#__206)
  3. 群控系統的物理樓層必須所有電梯設定一致[[225]](#__225)[[226]](#__226)
  4. 第一次自動運行前必須完成制動力測試[[226]](#__226)
